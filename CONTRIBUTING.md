## Before you open a PR / 提交 PR 前

- Keep changes focused and aligned with the specs referenced in the README.
  保持改动聚焦，并与 README 引用的规格一致。
- Run the gate locally: `pnpm lint && pnpm typecheck && pnpm test`.
  本地跑门禁：`pnpm lint && pnpm typecheck && pnpm test`。
- Describe what changed and why, and note any risks.
  说明改了什么、为什么，以及潜在风险。
